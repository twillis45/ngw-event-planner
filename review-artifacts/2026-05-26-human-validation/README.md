## How to use this kit

1. Read `00_plan/VALIDATION_EXECUTION_MATRIX.md` once before any session.
2. Open `01_session_kit/REPRODUCIBLE_URLS.md` in a browser tab during prep — bookmark the URLs.
3. For each operator:
   - Copy `01_session_kit/OBSERVATION_FORM.md` into `0X_operator_NNN/OBSERVATION_FORM.md`.
   - Fill it in DURING the session.
   - After the session, press Ctrl/Cmd+Shift+L and save the transcript to `06_observer_transcripts/operator_NNN_<slice>_<timestamp>.txt`.
4. After ≥ 2 operators, fill in `05_aggregated/AGGREGATE_FINDINGS.md`.
5. Paste the aggregate matrix into chat for synthesis.

## Why I (Claude) cannot run these sessions

Real operator cognition requires real operators. Fabricating session output —
hesitation timings, verbatim quotes, body-language reads — would be the exact
"fake validation" prohibited by the project's standing instruction. Infrastructure
prep, capture-kit construction, and post-session synthesis are mine. The sessions
themselves are yours.
