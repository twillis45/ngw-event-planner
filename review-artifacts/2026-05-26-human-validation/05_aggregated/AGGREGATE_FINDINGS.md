## Anti-pattern reminder

The strongest urge after a test is "just brighten the red" / "just add a pulse" /
"just make the button bigger." **Resist it.** Those instincts are exactly what the
Studio Matte doctrine exists to refuse. If structure isn't winning, fix structure.
Never saturation. Never motion drama.
